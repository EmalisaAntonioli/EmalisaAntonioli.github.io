Internship

Reverse Pulse Plate Automation

Summary

For my bachelor thesis internship at Cercuits, I developed a modular, user-friendly desktop application to control and monitor Reverse Pulse Plating (RPP) devices used in PCB manufacturing. The project involved integrating Modbus communication over RS485 with a Python-based PySide6 GUI, designed for easy operation by diverse user profiles within a production environment. Key features include centralized device management, dynamic program profiles, real-time monitoring with automated error detection, and role-based access control. The backend architecture follows a clean MVC pattern with event-based signaling and modular services, ensuring future scalability and maintainability. This project deepened my experience in industrial automation, serial communication, and software architecture, while emphasizing the value of detailed functional analysis and modular design in developing robust, production-ready applications.
